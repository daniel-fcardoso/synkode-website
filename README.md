# Synkode Website

Landing page desenvolvida para apresentar a proposta do projeto **Synkode**, uma plataforma focada em tornar o aprendizado de programação mais acessível, intuitivo e envolvente.

---

## 📌 Sobre o projeto

O **Synkode Website** foi criado como uma página de apresentação da ideia da plataforma, com o objetivo de comunicar sua proposta, identidade visual e principais funcionalidades.

A ideia central da Synkode é transformar o aprendizado de programação em uma experiência progressiva e interativa, inspirada em sistemas gamificados.

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## ⚙️ Funcionalidades

- Alternância entre modo claro e escuro  
- Menu responsivo para dispositivos móveis  
- Animações de entrada ao rolar a página  
- Formulário de contato com feedback ao usuário  
- Navegação entre seções da página  

---

## 🎯 Objetivo

Este projeto foi desenvolvido com foco em:

- Praticar desenvolvimento web  
- Consolidar conceitos de HTML, CSS e JavaScript  
- Criar um projeto com identidade própria  
- Dar os primeiros passos na construção da ideia da Synkode  

---

## 🧠 Aprendizados

Durante o desenvolvimento, foram aplicados conceitos como:

- Estruturação de páginas com HTML  
- Estilização e responsividade com CSS  
- Manipulação do DOM com JavaScript  
- Organização de layout e experiência do usuário  
- Implementação de interações simples
